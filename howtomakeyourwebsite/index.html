沈涵的网站  

如何创建你自己的网站

 1.关闭浏览器的安全防护

 2.打开c:\\windows\system32\drivers\etc\hosts

 3.打开cmd,输入ping github.com

 4.把获取到的ip在hosts文件中另起一行写成ip github.com的形式

 5.在cmd中输入ipconfig /flushdns并回车

 6.(如果你有邮箱,跳过此步;如果你有github账号,跳过6~10步)打开temp-mail.org,复制得到的邮箱

 7.打开github,点击sign up

 8.复制(或输入你自己的)邮箱,设置用户名和不少于15位的密码

 9.进行人机验证

 10.收验证码,如果没有显示输入验证码的框就直接返回主页

 11.回到github.com主页,点击sign in,输入用户名与密码登录

 12.点击右上角加号,点击new repository

 13.命名为"你的用户名.github.io",选择public,点击add a readme file

 14.进入这个repository,找到settings-pages-source,改成deploy from a branch

 15.再找到branch,改成main/(root),点击save

 16.返回该repository,点击add file-create new file

 17.编辑代码,可以设置分页名,将名字设置成index.html,点击commit changes

 18.等待30秒~10分钟,在浏览器中访问"你的用户名.github.com(/分页)"即可打开你的网站
